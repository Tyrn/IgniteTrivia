# IgniteTrivia

_The app is built according to Robin Heinze's tutorial,
[GitHub](https://github.com/robinheinze/ignite-trivia),
[Part 1](https://shift.infinite.red/creating-a-trivia-app-with-ignite-bowser-part-1-1987cc6e93a1),
[Part 2](https://shift.infinite.red/creating-a-trivia-app-with-ignite-bowser-part-ii-a286a978d0c8)_

- `npx ignite-cli new IgniteTrivia --remove-demo`, my arbitrary choices:

```
    Ignited IgniteTrivia in 76s  🚀

    For next time, here are the Ignite options you picked:
      npx ignite-cli new IgniteTrivia \
        --bundle=com.ignitetrivia \
        --git \
        --install-deps \
        --packager=npm \
        --target-path=/home/alexey/spaces/rn/IgniteTrivia \
        --remove-demo \
        --new-arch \
        --experimental=expo-router \
        --workflow=cng \
        --no-timeout=false \
        --state=mst
```

Note new-arch, expo-router, workflow, mst.

- Start the app on an Android device via USB cable:

```
npx expo run:android
```
